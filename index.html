CA MasterTest Website Launch Guide

Step-by-Step Instructions to Get Your Website Live

Completed Steps: 0/5

1
Save Your Website File

Copy the HTML code and save it as index.html on your computer.

Make sure the file has the .html extension.

Note: You can use Notepad (Windows) or TextEdit (Mac) to create the file. Just make sure to select "All Files" when saving and add the .html extension.

2
Create GitHub Account

Go to GitHub.com and create a free account.

GitHub will host your website for free with GitHub Pages.

Note: Choose a username that you might want to use in your website URL (e.g., github.com/yourusername/camastertest)

3
Create a New Repository

After logging in to GitHub:

  1. Click the "+" icon in the top right and select "New repository"
  2. Name it camastertest (or similar)
  3. Make sure it's set to Public
  4. Check "Add a README file"
  5. Click "Create repository"
Note: The repository name will be part of your website URL, so choose something professional and easy to remember.

4
Upload Your Website File

In your new repository:

  1. Click "Add file" â†’ "Upload files"
  2. Drag and drop your index.html file
  3. Add a commit message like "Add website file"
  4. Click "Commit changes"
Note: You can upload additional files like images later, but for now, just focus on getting the HTML file online.

5
Enable GitHub Pages

Now enable GitHub Pages to make your website live:

  1. Go to your repository's Settings tab
  2. Click Pages in the left sidebar
  3. Under "Source", select main branch
  4. Click Save
  5. Wait a few minutes, then refresh the page
  6. You'll see a message with your live website URL!
Note: Your website will be available at:
https://yourusername.github.io/camastertest/
